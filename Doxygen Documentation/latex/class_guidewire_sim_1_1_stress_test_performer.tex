\doxysubsection{Detailed Description}
This class enables the user to test the impact of mutliple external forces and external torques with one button within the Unity inspector. \begin{DoxyAttention}{Attention}
In the current version, the user is not able to fix positions or orientations of the guidewire, which is necessary e.\+g. for stress test one. 
\end{DoxyAttention}

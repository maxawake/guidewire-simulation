\doxysubsection{Detailed Description}
Debug Extension 


\begin{DoxyItemize}
\item Static class that extends Unity\textquotesingle{}s debugging functionallity.
\item Attempts to mimic Unity\textquotesingle{}s existing debugging behaviour for ease-\/of-\/use.
\item Includes gizmo drawing methods for less memory-\/intensive debug visualization.  
\end{DoxyItemize}
